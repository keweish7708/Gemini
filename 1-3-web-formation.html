    Web Formation Equipment

    Web forming systems of nonwoven production line

    As a critical step in web formation, the single-layer fiber web is cross-lapped or further drafted in order to form a fiber web with a certain thickness and basis weight, which can then undergo the next step of reinforcement and shaping.

    • Medium Speed Cross Lapper

      Cross lapper is used in various nonwoven production processes and allows to set the thickness, weight, and evenness of the final products. Here in cross lapper the fiber web becomes overlapped and ready for the next production process.

      Input width: ≤2500mm
      Output width: 2500mm~9000mm
      Infeed speed (horizontal): ≤40m/min
    • High Speed Cross Lapper

      High speed cross lapper is used in different nonwoven manufacturing processes such as needle punching, thermal bonding, chemical bonding, and spunlace.

      Input width: ≤2700mm
      Output width: 2500mm~7000mm
      Infeed speed: ≤60m/min, ≤80m/min
    • Vertical Lapper

      Vertical lapper is particularly suitable for indoor decoration nonwoven materials, insulation nonwoven materials, construction nonwoven materials, etc.

      Input width: 1500mm-2500mm
      Input speed: ≤40m/min
      Thickness adjustment: 50mm~2000mm
    • Air Laid Machine

      Airlaid machine utilizes aerodynamic methods to directly form fiber webs, which possess the advantages of random orientation and high uniformity. Airlaid machine can process diverse short fibers in the length from 20mm to 50mm, which is an ideal process for web formation of recycled or shoddy fiber.

      Capacity: 500-800kg/h
      Output width: 1000mm-2600mm
      Available products: 500-3000gsm
    • Web Drafter

      Web drafter is used to adjust the longitudinal distribution and orientation of the fiber web to improve the ratio of longitudinal and lateral strength to be similar, and it can also adjust the thickness and density of the fiber web to enhance production yield in conjunction with a nonwoven production line.

      Number of roller: 5 roller, 9 roller, 15 roller, 21 roller
      Working width: 2000mm~7000mm
